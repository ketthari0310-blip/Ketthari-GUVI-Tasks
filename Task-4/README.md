# Income & Expense Tracker

A simple **CRUD-based expense tracker** built with **HTML, CSS, and JavaScript**.  
It allows users to add, edit, delete, and filter income/expense entries, with data persisted in **local storage**.

## Features
- Add new income and expense entries
- Edit existing entries
- Delete entries
- Filter by "All", "Income", or "Expense"
- Display total income, total expenses, and net balance
- Reset button to clear input fields
- Local storage persistence
- Responsive design for desktop and mobile

## How to Use
1. Open `index.html` in your browser.
2. Enter a description and amount, select type (income/expense), and click **Add Entry**.
3. Use filters to view specific entries.
4. Edit or delete entries directly from the list.
5. Totals update automatically.

## Tech Stack
- HTML5
- CSS3 (Responsive Design)
- Vanilla JavaScript (CRUD + Local Storage)
